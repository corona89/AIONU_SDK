## 시스템 요구사항 및 제약
1. **브라우저 지원**: IE11, Chrome, Edge 지원.
2. **기술 스택**: JavaScript, jQuery 기반 (특정 프레임워크 종속 방지).
3. **설정 관리**: Endpoint 및 API Key는 설정 파일로 관리.
4. **독립성**: jQuery 파일을 내장(vendor)하여 제공 (최신 버전).
